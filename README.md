# THREAD

Distributed transaction tracing SDK + AI-powered recovery platform built for the Splunk Agentic Ops Hackathon 2026.

THREAD automatically correlates logs across microservices via shared Correlation IDs, stores failed transactions for replay, and deploys an AI agent via Splunk MCP Server to investigate failures, set a dynamic replay limit L, and deliver one-click recovery to the ops team via Slack

## Traceable Header Relay, Execution Audit & Distributed-recovery

Built for the Splunk Agentic Ops Hackathon 2026.

**Secrets** (tokens, passwords, API keys) are managed via [Infisical](https://infisical.com).

Run all commands prefixed with `infisical run --`.

**Config** (ports, URLs, flags) lives in `.env` — see `.env.example`.
